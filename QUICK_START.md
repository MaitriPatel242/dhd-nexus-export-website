## Easiest Option: VS Code Live Server (Recommended)

If you're using VS Code:

1. **Install Live Server Extension:**
   - Open VS Code
   - Click the Extensions icon (or press `Ctrl+Shift+X`)
   - Search for "Live Server" by Ritwick Dey
   - Click "Install"

2. **Start the Server:**
   - Right-click on `contact.html` in VS Code
   - Select **"Open with Live Server"**
   - Your browser will open automatically at `http://127.0.0.1:5500/contact.html`

3. **Test the Form:**
   - Fill out and submit the contact form
   - Check `maitri240205@gmail.com` for the email

**That's it!** The server runs automatically and the form will work.

---

## Alternative: Use Node.js (If you have it)

If you have Node.js installed:

1. Open Terminal/Command Prompt in this folder
2. Run: `npx http-server -p 8000`
3. Open: `http://localhost:8000/contact.html`

## Alternative: Online Testing

If you want to test without installing anything:

1. Upload your website to a free hosting service:
   - **Netlify Drop**: Drag and drop your folder to [netlify.com/drop](https://app.netlify.com/drop)
   - **GitHub Pages**: Push to GitHub and enable Pages
   - **Vercel**: Upload your folder

2. Once live, the form will work automatically!
