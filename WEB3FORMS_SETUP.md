# Web3Forms Setup - Quick Guide

Web3Forms is more reliable than FormSubmit and **doesn't require activation**!

## Step 1: Get Your Free Access Key

1. Go to: **https://web3forms.com**
2. Enter your email: **maitri240205@gmail.com**
3. Click **"Get Your Access Key"**
4. Copy the access key (it will look like: `a1b2c3d4-e5f6-7890-abcd-ef1234567890`)

## Step 2: Update the Form

1. Open `contact.html` in Cursor
2. Find this line (around line 103):
   ```html
   <input type="hidden" name="access_key" value="YOUR_ACCESS_KEY" />
   ```
3. Replace `YOUR_ACCESS_KEY` with the key you copied
4. Save the file

## Step 3: Test

1. Make sure Live Server is running in Cursor
2. Open the contact form
3. Fill out and submit
4. Check `maitri240205@gmail.com` for the email
